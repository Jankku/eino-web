## Run project

- Rename `.env.example` file to `.env`.
- In `.env` file change `REACT_APP_BASE_URL` value to point to Eino backend. 
- Install dependencies:
```
npm install
```
- Start development server:
```
npm start
```

## Build
Build production version with the following command:
```
npm run build
```
